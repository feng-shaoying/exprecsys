| 23/4/2022    | Exploring Relations between Personality and User Rating Behaviors | Rong Hu, Pearl Pu | 2013 |          | 研究性格特征与用户评分行为之间的关系 | 1. 研究用户性格特征对用户评分行为的影响，包括评分的数量、正面评分的数量、用户评分的类别覆盖率和用户评分的兴趣多样性，并考虑用户的年龄、性别和用户评分行为的经验。<br/>s2. 沿着用户评分的长尾探索性格差异。 | 1. BFI44项问卷调查。2. 对问卷调查结果进行相关性分析。 | \1. 责任心低的人倾向于评分更多的项目，而责任心高的人倾向于只评分所要求的项目数量。同样地，经验开放性也会影响用户评分项目的数量。然而，他们更有可能在一定范围内对更多的物品进行评分，可能是为了满足他们的好奇心。超过这个界限，他们将停止评分，而低尽责性的人将继续评分更多项目。这一发现让我们重新思考用户评分的有效性，以及如何找到这些有效评分。这可能是用户建模领域的一项有趣的研究。<br/>\2. 以往的研究表明，亲和性与好友数量、群组数量和“点赞”数量呈正相关。我们的研究结果一致表明，亲和性高的个体倾向于给予更积极的评分。这意味着我们很难知道高亲和性用户的实际偏好或意见。因此，由于评分的妥协，利用评分来推断他们的兴趣是存在风险的。<br/>\3. 责任心对类别覆盖面和兴趣多样性有负向影响。然而，那些责任心高的人倾向于在有限的类别中进行项目评分，他们的评分可能专注于某些类别。多样性是信息检索和推荐系统领域的一个研究热点。似乎在责任心方面得分低的人要比得分高的人更容易行成多样化。考虑到评分的有效性问题，这种多样化是否会对个性化过程有利还不得而知。另一个研究问题是，责任心得分高的用户是否真的喜欢缩小的项目范围，以及如何帮助他们更多样化地打分。<br/>\4. 性别另一个在统计上与评分数量、类别覆盖面和兴趣多样性变量显著相关的因素。 | [link](/Personality/Hu_Pu_2013_ExploringRelationsbetweenPersonalityandUserRatingBehaviors.md) |
